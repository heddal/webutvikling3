### howto Express

prøv først uten denne:
```
$ npm install
```
og bare disse:
```
$ npm install express --save
```
```
$ npm install --save-dev nodemon
```
```
$ npm install concurrently --save
```
### For testing
```
npm install jest --save-dev
npm install cypress --save-dev
```
run tests with:
```
node_modules/.bin/cypress open
```
